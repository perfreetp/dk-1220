## 1. 产品概述

服务拓扑观察台是一个面向研发负责人和运维值班人员的 Web 应用，用于可视化展示和管理业务系统之间的服务依赖关系。通过自动绘制调用关系图、实时监控服务状态、关联告警信息、追踪变更记录等功能，帮助团队快速定位故障影响范围、评估系统风险、支持运维决策。

**核心价值**：提升系统可观测性，缩短故障定位时间，降低运维沟通成本。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 研发负责人 | 查看拓扑、管理服务信息、标记核心服务、查看变更记录、导出快照 |
| 运维值班人员 | 查看拓扑、监控告警、查看异常链路、生成故障影响清单、导出快照 |

### 2.2 功能模块

1. **拓扑总览页面**：服务拓扑图可视化、节点筛选、依赖关系展示
2. **服务详情页面**：服务信息管理、上下游依赖查看、接口列表
3. **链路查询页面**：接口影响范围搜索、异常链路展示
4. **变更记录页面**：发布变更记录、变更历史查询
5. **告警面板页面**：告警列表、告警关联服务、故障影响清单

## 3. 核心流程

### 3.1 服务录入与拓扑生成流程

用户录入服务信息（服务名称、负责人、业务域、环境、接口列表）→ 系统解析接口调用关系 → 自动生成服务依赖边 → 渲染拓扑关系图 → 支持筛选和搜索

### 3.2 故障定位流程

告警触发 → 用户查看告警面板 → 点击告警定位服务 → 查看上下游依赖 → 分析影响范围 → 生成故障影响清单 → 导出快照用于沟通

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色科技风（深蓝 #0A1628 背景 + 青色 #00D9FF 强调色 + 橙色 #FF6B35 告警色）
- **辅助色**：绿色 #10B981（正常状态）、黄色 #F59E0B（警告）、红色 #EF4444（异常）
- **按钮风格**：圆角矩形（8px），微光效果，悬停时发光
- **字体**：标题使用 JetBrains Mono（技术感），正文使用 Inter
- **布局风格**：左侧导航栏 + 顶部工具栏 + 主内容区，卡片式布局
- **图标风格**：线性图标，2px 描边，与文字颜色一致
- **动效**：节点悬停发光、连线流动动画、页面切换淡入淡出

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 拓扑总览 | 拓扑图画布 | 深色背景，节点为圆角矩形卡片，连线带流动动画，支持缩放平移 |
| 拓扑总览 | 筛选面板 | 左侧固定面板，下拉选择器，标签式筛选器 |
| 拓扑总览 | 工具栏 | 顶部工具栏，包含搜索框、缩放控制、导出按钮 |
| 服务详情 | 信息卡片 | 右侧抽屉式面板，表单输入框，标签选择器 |
| 服务详情 | 依赖视图 | 迷你拓扑图，上游绿色节点，下游蓝色节点 |
| 链路查询 | 搜索区域 | 大型搜索框，自动补全，搜索历史 |
| 链路查询 | 结果展示 | 树形结构展示，异常节点红色高亮 |
| 变更记录 | 时间线 | 左侧时间轴，右侧变更卡片，按日期分组 |
| 告警面板 | 告警卡片 | 卡片列表，严重程度颜色标识，时间戳，快捷操作按钮 |

### 4.3 响应式设计

- **桌面优先**：主要针对大屏幕（1920x1080 及以上）优化，支持多窗口并排查看
- **平板适配**：侧边栏可折叠，拓扑图自适应缩放
- **触摸优化**：支持触摸拖拽、双指缩放

### 4.4 交互细节

- **拓扑图交互**：
  - 节点悬停显示服务状态浮层
  - 点击节点展开详情抽屉
  - 拖拽节点调整布局
  - 双击节点聚焦并高亮上下游
  - 连线悬停显示调用详情

- **告警联动**：
  - 告警面板点击告警，拓扑图自动定位并高亮
  - 异常服务节点闪烁动画
  - 告警数量徽标显示在导航图标上

- **数据可视化**：
  - 核心服务节点加大尺寸并添加星标
  - 服务状态通过节点颜色和边框表示
  - 调用频率通过连线粗细表示
